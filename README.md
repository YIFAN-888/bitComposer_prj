# bit composer
 自動作曲Webアプリケーション

# アプリの機能・特徴
ユーザが作った短いメロディからゲーム音楽風の曲を作曲してくれる

# 使い方
1. Webページ上に表示されるピアノからメロディを入力していく
2. メロディをサーバに送信してAIが作曲
3. 曲のmp3を再生

# Webシステムについて
サーバプログラムはPythonのライブラリのFlaskを使っています。
